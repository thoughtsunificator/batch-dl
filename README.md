# batch-dl

CLI to download a batch list of URLs.

## Getting Started

### Install

### Usage

- ``npx @thoughtsunificator/batch-dl [input-file.txt]``

### Input file format

Text file containing a list of URLs separated by a new line character.

* Only supports HTTP and HTTPS protocols.

```
https://.../file.bin
https://.../file2.bin
https://.../file3.bin
https://.../file4.bin
https://.../file5.bin
```
